# 终端钢琴演奏功能

## 架构概览

```
用户：帮我弹一首《小星星》
   ↓
Daily Agent 收到请求
   ↓
调用 play_piano 工具（传入 song_name）
   ↓
LLM 实时编曲 → 返回音符序列 + 节拍信息
   ↓
Ink 渲染钢琴 UI（覆盖层）
   ↓
按节拍逐个高亮琴键 + 播放音调
   ↓
演奏完毕，自动退出钢琴 UI
```

## 第一步：音频引擎（src/tools/piano/audio-engine.ts）

**目标**：用 Web Audio API 或 `speaker` + `pcm` 生成并播放正弦波音调。

**技术方案**：使用 `speaker` npm 包 + 手动生成 PCM 正弦波数据（无需预生成 wav 文件）。

```typescript
// 音符频率映射表（C4-B5，两个八度）
const NOTE_FREQS: Record<string, number> = {
  'C4': 261.63, 'C#4': 277.18, 'D4': 293.66, 'D#4': 311.13,
  'E4': 329.63, 'F4': 349.23, 'F#4': 369.99, 'G4': 392.00,
  'G#4': 415.30, 'A4': 440.00, 'A#4': 466.16, 'B4': 493.88,
  'C5': 523.25, // ... 扩展到 C6
};

// 生成指定频率和时长的 PCM 数据（16-bit, 44100Hz）
function generateSineWave(frequency: number, durationMs: number): Buffer {
  const sampleRate = 44100;
  const numSamples = Math.floor(sampleRate * durationMs / 1000);
  const buffer = Buffer.alloc(numSamples * 2); // 16-bit = 2 bytes
  
  for (let i = 0; i < numSamples; i++) {
    const t = i / sampleRate;
    const sample = Math.sin(2 * Math.PI * frequency * t);
    const int16 = Math.max(-1, Math.min(1, sample)) * 0x7FFF;
    buffer.writeInt16LE(int16, i * 2);
  }
  
  return buffer;
}

// 播放音符
async function playNote(note: string, durationMs: number): Promise<void> {
  const freq = NOTE_FREQS[note];
  if (!freq) return;
  
  const pcm = generateSineWave(freq, durationMs);
  const speaker = new Speaker({ sampleRate: 44100, channels: 1, bitDepth: 16 });
  
  return new Promise((resolve) => {
    speaker.on('finish', () => resolve());
    speaker.end(pcm);
  });
}
```

**依赖**：
```bash
npm install speaker
npm install --save-dev @types/speaker
```

## 第二步：LLM 编曲工具（src/tools/piano/composer.ts）

**目标**：通过 LLM 将歌曲名转换为音符序列 + 节拍信息。

```typescript
export interface NoteEvent {
  note: string;       // 如 'C4', 'E4', 'G4'
  duration: number;   // 毫秒，如 400（四分音符）、200（八分音符）
  isRest?: boolean;   // 是否为休止符
}

export async function composeSong(
  client: ILLMClient,
  songName: string
): Promise<{ title: string, notes: NoteEvent[] }> {
  const prompt = `将歌曲「${songName}」的主旋律转换为简谱音符序列。

要求：
1. 只返回 JSON，不要其他内容
2. 使用 C 大调，音域限制在 C4-B5（两个八度）
3. 节拍：四分音符=400ms，八分音符=200ms，二分音符=800ms
4. 返回格式：
{
  "title": "歌曲名",
  "notes": [
    {"note": "C4", "duration": 400},
    {"note": "C4", "duration": 400},
    {"note": "G4", "duration": 400},
    {"note": "G4", "duration": 400}
  ]
}

只返回 JSON，不要 markdown 代码块包裹。`;

  const response = await client.ask(prompt, '你是音乐编曲助手，擅长将歌曲转换为简谱音符序列。');
  
  // 解析 JSON 响应
  const jsonMatch = response.match(/\{[\s\S]*\}/);
  if (!jsonMatch) throw new Error('LLM 返回格式无效');
  
  return JSON.parse(jsonMatch[0]);
}
```

## 第三步：钢琴 Ink 组件（src/cli/ui/components/Piano.tsx）

**目标**：渲染 2 个八度的钢琴键盘（C4-B5），支持高亮指定琴键。

```tsx
interface PianoProps {
  activeNote: string | null;  // 当前高亮的音符，如 'C4'
  songTitle: string;
  progress: number;            // 演奏进度 0-100
}

export function Piano({ activeNote, songTitle, progress }: PianoProps) {
  const whiteKeys = ['C', 'D', 'E', 'F', 'G', 'A', 'B'];
  const blackKeys = ['C#', 'D#', null, 'F#', 'G#', 'A#', null];
  const octaves = [4, 5];

  // 渲染白键行
  const renderWhiteKeys = () => (
    <Box>
      {octaves.map(octave =>
        whiteKeys.map(note => {
          const fullName = `${note}${octave}`;
          const isActive = activeNote === fullName;
          return (
            <Box
              key={fullName}
              width={6}
              height={8}
              borderStyle="single"
              borderColor={isActive ? 'yellow' : 'gray'}
              backgroundColor={isActive ? 'yellow' : 'white'}
              justifyContent="center"
              alignItems="center"
            >
              <Text color={isActive ? 'black' : 'gray'} bold={isActive}>
                {note}{octave}
              </Text>
            </Box>
          );
        })
      )}
    </Box>
  );

  // 渲染黑键行（交错在白键上方）
  const renderBlackKeys = () => (
    <Box paddingLeft={3}>
      {octaves.map(octave =>
        blackKeys.map((note, i) => {
          if (!note) return <Box key={`empty-${octave}-${i}`} width={6} />;
          const fullName = `${note}${octave}`;
          const isActive = activeNote === fullName;
          return (
            <Box
              key={fullName}
              width={4}
              height={5}
              borderStyle="single"
              borderColor={isActive ? 'yellow' : 'black'}
              backgroundColor={isActive ? 'yellow' : 'black'}
              justifyContent="center"
              alignItems="center"
            >
              <Text color={isActive ? 'black' : 'white'} bold={isActive}>
                {note.replace('#', '♯')}
              </Text>
            </Box>
          );
        })
      )}
    </Box>
  );

  return (
    <Box flexDirection="column" marginY={1}>
      <Box justifyContent="space-between">
        <Text bold color="cyan">🎹 正在演奏: {songTitle}</Text>
        <Text dimColor>{progress}%</Text>
      </Box>
      {renderBlackKeys()}
      {renderWhiteKeys()}
      <Text dimColor>ESC 停止演奏</Text>
    </Box>
  );
}
```

## 第四步：钢琴播放工具（src/tools/piano/play-piano.ts）

**目标**：组合 LLM 编曲 + 音频播放 + UI 渲染。

```typescript
export const definition: ToolDefinition = {
  type: 'function',
  function: {
    name: 'play_piano',
    description: '在终端中演奏钢琴动画。根据歌曲名实时编曲，渲染钢琴键盘 UI，按旋律高亮琴键并播放音调。适合用户要求弹钢琴、演奏歌曲时使用。',
    parameters: {
      type: 'object',
      properties: {
        song_name: {
          type: 'string',
          description: '要演奏的歌曲名，如 "小星星"、"欢乐颂"、"生日快乐"',
        },
      },
      required: ['song_name'],
    },
  },
};

export const implementation: ToolFunction = async (args: { song_name: string }) => {
  // 1. 通过 store 推送 piano-ui 事件，让 App 渲染钢琴界面
  // 2. LLM 编曲
  const song = await composeSong(client, args.song_name);
  
  // 3. 逐个播放音符，同时更新 UI 高亮
  for (let i = 0; i < song.notes.length; i++) {
    const note = song.notes[i];
    const progress = Math.round(((i + 1) / song.notes.length) * 100);
    
    // 推送事件更新 UI
    store.push({
      type: 'piano-update',
      activeNote: note.note,
      songTitle: song.title,
      progress,
    });
    
    // 播放音调
    await playNote(note.note, note.duration);
    
    // 音符间短暂停顿
    await sleep(50);
  }
  
  // 4. 演奏完毕，推送结束事件
  store.push({ type: 'piano-end' });
  return `演奏完成：${song.title}`;
};
```

const [pianoState, setPianoState] = useState<{
  visible: boolean;
  activeNote: string | null;
  songTitle: string;
  progress: number;
}>({ visible: false, activeNote: null, songTitle: '', progress: 0 });

// 订阅钢琴事件
useEffect(() => {
  const unsubscribe = store.subscribePiano(state => {
    setPianoState(state);
  });
  return unsubscribe;

// 在渲染中

  <Piano
    activeNote={pianoState.activeNote}
    songTitle={pianoState.songTitle}

## 技术要点

1. **音频播放不阻塞 UI**：使用 `speaker` 的流式播放，音符播放和 UI 更新同步进行
2. **ESC 中断**：在 `useInput` 中监听 ESC，设置中断标志，`playNote` 循环中检查并提前退出
3. **LLM 编曲容错**：LLM 返回可能不标准，需要 JSON 解析 + fallback 到预设曲目
4. **音调优化**：正弦波添加 ADSR 包络（Attack-Decay-Sustain-Release）让音色更接近真实钢琴
